# docker入门

### 概述

本仓库用来记录docker入门笔记！

docker与虚拟机的区别：

- 虚拟机：与宿主机共享硬件，支持安装与宿主机不同的操作系统；
- docker：与宿主机共享操作系统（文件分层，共享bootfs），只支持安装与宿主机相同操作系统的操作系统，即Linux宿主机支持Linux一系列操作系统。
