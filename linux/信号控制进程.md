常用信号：

- 1：重新加载配置，PID不变，用于更新配置文件的情况
- 9：强制终止
- 15：安全终止，会进行资源的释放
- 18：继续
- 19：停止
- 20：暂停 CTRL + z

常用命令：

- kill 信号 PID/Job ID
- killall 信号 name
- pkill 选项 模式：给进程发送消息
- w：显示远程连接的用户
- ps aux | grep sshd：显示远程连接的用户
- pgrep -u username sshd：显示某个进程的PID
